## Встановлення пакетів через npm

```bash
$ npm install
```

## Запуск проєктів

```bash
# code-review
$ npm run start:dev:code-review

# main
$ npm run start:dev:main

# telegram-bot
$ npm run start:dev:telegram-bot
```

## Запуск мікро-сервісів

```bash
# user microservice
$ npm run start:dev m-user

# token microservice
$ npm run start:dev m-token
```
